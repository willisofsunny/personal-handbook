## 🚀 快速開始 / Quick Start

1. **下載文件** / Download Files
   - 將所有文件下載到本地資料夾

2. **開啟網頁** / Open Website
   - 雙擊 `index.html` 文件
   - 或在瀏覽器中開啟 `index.html`

3. **切換語言** / Switch Language
   - 點擊右上角的語言切換按鈕
   - 支援繁體中文、簡體中文、英文

## 🔧 自訂化 / Customization

### 修改內容 / Content Modification
1. 編輯 `index.html` 中的文字內容
2. 更新 `data-zh-TW`、`data-zh-CN`、`data-en` 屬性來修改多語言內容

### 修改樣式 / Style Modification
1. 編輯 `styles.css` 文件
2. 主要變數位於文件頂部

### 修改功能 / Function Modification
1. 編輯 `script.js` 文件
2. 每個功能都是獨立的類別
